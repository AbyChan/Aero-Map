# Aero-Map
Rebirth from [socialMap](https://github.com/vtch99/SocialMap).

地理社交应用，毕业设计。
#技术参数：
-   leafet地图
-   cordova App
-   前后段分离
-   angular
-   postgre地理数据支持
-   node.js
-   bookshelf ORM
-   redis Cache
-   socket.io
-   foundation ui

#功能
-   注册登陆等
-   搜索附近的人
-   在地图上发表标记博文。支持图片上传
-   搜索附近标记博文
-   博文管理
-   好友系统
-   即时聊天
